# Problem Statement
Managing a library manually is time-consuming and error-prone. The Smart Library Management System aims to automate library operations, making it easier to manage books, users, and transactions efficiently.

# Scope
- Manage users and roles
- Maintain book inventory
- Handle book issue/return
- Generate reports

# Target Users
- Students
- Librarians
- Administrators

# High-Level Features
- User registration and authentication
- Book search and management
- Transaction management (issue/return)
- Fine calculation
- Reporting and analytics

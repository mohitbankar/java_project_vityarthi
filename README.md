# Smart Library Management System

## Overview
A Java-based application to manage library resources, user accounts, and book transactions, with analytics and reporting features.

## Features
- User management (students, librarians, admin)
- Book inventory management
- Book issue/return and fine calculation
- Reporting and analytics

## Technologies/Tools Used
- Java
- Collections framework
- File I/O (or SQLite)
- MVC architecture
- JUnit (for testing)
- Git

## Installation & Run Steps
1. Clone the repository
2. Open a command prompt and navigate to the project root directory
3. Compile the Java source files:
   ```cmd
   cd src
   javac library\*.java
   ```
4. Run the application:
   ```cmd
   java library.Main
   ```

**Note:**
- Ensure Java (JDK) is installed and `javac`/`java` are available in your PATH.
- If you encounter errors, check that your `Main.java` file starts with `package library;`
